Search Seal Descriptions

Seal descriptions search enables users to access the entries in published catalogues, card catalogues, finding aids and other such reference works that describe seals. In some cases, there are multiple entries for a single seal, as some seals have been catalogued separately multiple times. A user can limit the seal description search results using three filters.

  • Reference work
  • Entry Identifier
  • Entry Title

Search Actors

Actor search enables users to find people and corporate entities that had seals. A user can limit the actor search results using four filters.

  • Name
  • Actor Type
  • Corporate Actor Class
  • Corporate Actor Order

Search Places

Place search enables users to find places that are associated with seals and to limit the search results using two filters.

  • County
  • Place Name
